EarnTube
Earn money by posting content platform with user dashboard and admin panel. Users earn coins through likes, shares, and referrals.

Features
User registration & login

Post creation with images

Like/comment/share system

Coin earning system

Referral program (5 PKR per referral)

Withdrawals to JazzCash/EasyPaisa

Admin dashboard for management

Real-time chat

Mobile responsive

Files
index.html - Main user interface

admin.html - Admin management panel

Tech Used
HTML, CSS, JavaScript, Firebase

Setup
Upload both HTML files to GitHub

Enable GitHub Pages in repository settings

Configure Firebase in console

Add admin credentials in Firebase
